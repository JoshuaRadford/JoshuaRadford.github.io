Procedural Map Generator

Delaunay Triangulation: A Pedagogical Aid

This project is a web-based tool to demonstrate the delaunay triangulation algorithm. The application will take a set of parameters from the user, utilizing three.js to generate an interactive terrain.

Presentation (5-weeks) | Presentation (10-weeks) | Presentation(15-weeks) | Final Presentation
  1. Confirm your chosen Terrain Configuration.
  2. Click Generate.
  3. Use the mouse to control the camera.
    1. Left-click: rotate the camera
    2. Right-click: translate the camera
    3. Scroll: zoom in/out the camera
    4. Double-click: reset the camera


  • Adjust the Noise Configuration
  • Toggle the (Wireframe / Texture) or
    (No Points / Show Points).
  • Toggle Edit Mode to add points on click.
